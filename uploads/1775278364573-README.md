### 1. `fetch()`

**`fetch()` क्या होता है?**
`fetch()` एक मॉडर्न JavaScript फंक्शन है जिसका इस्तेमाल वेब पेज से किसी दूसरे सर्वर या API (Application Programming Interface) से डेटा लाने (request करने) के लिए किया जाता है। यह एक तरह से आपके वेब पेज को "बाहर की दुनिया" से बात करने का तरीका देता है।

**कल्पना कीजिए:** आप एक रेस्टोरेंट में बैठे हैं और वेटर को ऑर्डर दे रहे हैं। `fetch()` बिल्कुल वैसा ही है। आपका वेब पेज (आप) `fetch()` (वेटर) को बताता है कि उसे क्या चाहिए (किस URL से डेटा चाहिए), और `fetch()` उस डेटा को लाने के लिए सर्वर (किचन) के पास जाता है।

**मुख्य बातें:**
*   **Asynchronous (अतुल्यकालिक):** इसका मतलब है कि जब `fetch()` डेटा लाने जाता है, तो आपका वेब पेज उसका इंतज़ार नहीं करता। वह अपना बाकी काम करता रहता है। जब डेटा आ जाता है, तब `fetch()` आपको बताता है।
*   **Promises:** `fetch()` एक 'Promise' रिटर्न करता है। Promise का मतलब है कि "मैं तुम्हें डेटा लाने का वादा करता हूँ। जब डेटा आ जाएगा (या कोई गलती होगी), तो मैं तुम्हें बता दूँगा।" आप `.then()` का उपयोग करके बताते हैं कि डेटा आने पर क्या करना है, और `.catch()` का उपयोग करके बताते हैं कि कोई गलती होने पर क्या करना है।

**उदाहरण:**
मान लीजिए आपको एक वेबसाइट से यूज़र्स की लिस्ट चाहिए।

```javascript
// fetch() का उपयोग करके डेटा लाना
fetch('https://jsonplaceholder.typicode.com/users') // इस URL से डेटा लाओ
  .then(response => {
    // जब डेटा आ जाए (response मिल जाए), तो उसे JSON फॉर्मेट में बदलो
    // (response.json() के बारे में नीचे और जानेंगे)
    return response.json();
  })
  .then(data => {
    // जब डेटा JSON फॉर्मेट में बदल जाए, तो उसे console में दिखाओ
    console.log('Users data:', data);
    // आप इस डेटा का उपयोग करके वेब पेज पर कुछ भी दिखा सकते हैं
  })
  .catch(error => {
    // अगर डेटा लाने में या बदलने में कोई गलती हो, तो उसे console में दिखाओ
    console.error('Error fetching data:', error);
  });

console.log('यह मैसेज fetch() के डेटा आने से पहले ही दिख जाएगा!');
// क्योंकि fetch() asynchronous है, यह लाइन तुरंत execute हो जाएगी।
// डेटा आने में थोड़ा समय लगेगा।
```

### 2. `JSON kya hota hai`

**JSON क्या होता है?**
JSON का पूरा नाम है **JavaScript Object Notation**। यह डेटा को स्टोर करने और एक जगह से दूसरी जगह भेजने का एक बहुत ही पॉपुलर, हल्का और टेक्स्ट-आधारित तरीका है।

**कल्पना कीजिए:** आप किसी दोस्त को अपनी पसंदीदा फिल्मों की लिस्ट भेजना चाहते हैं। आप उसे एक सादे टेक्स्ट में लिख सकते हैं, लेकिन अगर आप उसे एक स्ट्रक्चर्ड तरीके से लिखें ताकि वह आसानी से समझ सके और कंप्यूटर भी उसे आसानी से पढ़ सके, तो वह JSON है।

**मुख्य बातें:**
*   **टेक्स्ट-आधारित:** यह सिर्फ टेक्स्ट होता है, इसलिए इसे किसी भी प्रोग्रामिंग भाषा में पढ़ा और लिखा जा सकता है।
*   **मानव-पठनीय:** इसे इंसान भी आसानी से पढ़ और समझ सकते हैं।
*   **की-वैल्यू पेयर (Key-Value Pairs):** डेटा को `{"key": "value"}` के फॉर्मेट में स्टोर किया जाता है। जैसे: `"name": "Alice"`, `"age": 30`।
*   **ऑब्जेक्ट्स और एरेज़:** JSON ऑब्जेक्ट्स `{}` कर्ली ब्रेसेज़ में होते हैं और की-वैल्यू पेयर का कलेक्शन होते हैं। JSON एरेज़ `[]` स्क्वायर ब्रैकेट्स में होते हैं और वैल्यूज़ की लिस्ट होते हैं।
*   **डेटा टाइप्स:** JSON में स्ट्रिंग्स (strings), नंबर्स (numbers), बूलियन (booleans - true/false), एरेज़ (arrays), ऑब्जेक्ट्स (objects) और null जैसे डेटा टाइप्स हो सकते हैं।

**उदाहरण:**
एक व्यक्ति की जानकारी JSON फॉर्मेट में:

```json
{
  "name": "अमित कुमार",
  "age": 28,
  "isStudent": false,
  "hobbies": ["पढ़ना", "कोडिंग", "यात्रा"],
  "address": {
    "street": "123 मुख्य मार्ग",
    "city": "दिल्ली",
    "zipCode": "110001"
  },
  "email": null
}
```
यह डेटा किसी सर्वर से आ सकता है और आपका JavaScript कोड इसे आसानी से पढ़ सकता है।

### 3. `.json()` method

**`.json()` method क्या होता है?**
यह `fetch()` द्वारा रिटर्न किए गए `Response` ऑब्जेक्ट का एक मेथड है। इसका काम है `fetch()` से मिली रॉ (raw) नेटवर्क रिस्पांस को पढ़ना और उसे एक JavaScript ऑब्जेक्ट में पार्स (parse) करना।

**कल्पना कीजिए:** `fetch()` ने आपको एक लिफाफा (raw response) दिया है जिसमें JSON डेटा टेक्स्ट के रूप में लिखा हुआ है। `.json()` मेथड उस लिफाफे को खोलता है, टेक्स्ट को पढ़ता है, और उसे एक ऐसे फॉर्मेट में बदल देता है जिसे आपका JavaScript कोड सीधे इस्तेमाल कर सके (जैसे एक JavaScript ऑब्जेक्ट या एरे)।

**मुख्य बातें:**
*   यह भी एक **asynchronous** ऑपरेशन है और एक Promise रिटर्न करता है।
*   यह `fetch()` के `.then()` ब्लॉक के अंदर इस्तेमाल होता है।
*   यह सिर्फ JSON टेक्स्ट को ही JavaScript ऑब्जेक्ट में बदल सकता है।

**उदाहरण (ऊपर वाले `fetch()` उदाहरण का हिस्सा):**

```javascript
fetch('https://jsonplaceholder.typicode.com/users')
  .then(response => {
    // यहाँ 'response' एक Response ऑब्जेक्ट है, जिसमें raw डेटा है।
    // response.json() उस raw डेटा को JSON टेक्स्ट मानकर JavaScript ऑब्जेक्ट में बदलता है।
    return response.json(); // यह एक Promise रिटर्न करता है
  })
  .then(data => {
    // 'data' अब एक असली JavaScript ऑब्जेक्ट या एरे है, जिसे आप सीधे इस्तेमाल कर सकते हैं।
    console.log(data[0].name);
